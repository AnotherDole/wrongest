The Wrongest Words

a party game for 3-8 players

Visit the project page at wrongest.website to learn more.

This project was made by Dole and Lemon, with some help from the F Plus Ridiculists.

Project uses:

  • node.js
  • socket.io
  • redis
  • jQuery
  • jQuery Knob
  • OpenShift
